# 06 – Advanced Konzepte

In diesem Ordner sind weitere fortgeschrittene Git-Themen gesammelt:

- `01_reflog` – Commits „wiederfinden“, die scheinbar verloren sind.
- `02_bisect` – Commit finden, der einen Bug eingeführt hat.
- `03_cherry-pick-revert` – Einzelne Commits gezielt übernehmen oder rückgängig machen.
- `04_worktree_optional` – Mehrere Arbeitsverzeichnisse für ein Repo.

Jeder Unterordner enthält ein eigenes `README.md` und teilweise ein `setup.sh`.
