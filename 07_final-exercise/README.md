# 07 – Abschlussübung (Kombi-Übung)

In dieser Übung kombinierst du mehrere der gelernten Konzepte in einem etwas größeren Szenario.

## Setup

```bash
cd 07_final-exercise
./setup.sh
```

Dadurch entsteht ein Repo `repo-final-demo/` mit mehreren Branches und einer kleinen „Mini-App“.

## Ziel

Arbeite dich durch die Aufgaben in `tasks.md`. Versuche zuerst, sie ohne Blick in die Lösungen zu lösen. In `solutions.md` findest du einen möglichen Lösungsweg.

Diese Übung ist gut geeignet, um in Gruppen zu arbeiten.
